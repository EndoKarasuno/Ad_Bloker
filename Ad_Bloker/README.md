# JS版 簡易広告ブロッカー

指定されたURLのWebページを取得し、広告やトラッキング要素を無力化して、安全なサンドボックス環境内に表示するツールです。

## 機能

- **広告ブロック**: `div`や`iframe`などの広告コンテナを削除します。
- **追跡防止**: `sandbox`属性を利用し、Cookie、LocalStorage、各種トラッキングスクリプトの実行を強力にブロックします。
- **外部リンクの無効化**: 処理中のページから、意図せず他のサイトへ移動するのを防ぎます。
- **堅牢なページ取得**:
  - **文字コード自動判別**: `jschardet`ライブラリによる内容解析とHTTPヘッダーの情報を組み合わせ、Shift_JISやUTF-8などの文字コードを自動で判別し、文字化けを防ぎます。
  - **プロキシフォールバック**: 複数の公開CORSプロキシをリスト化し、一つが失敗しても自動で次を試すことで、通信の成功率を高めます。
- **レガシーサイト対応**: `frameset`で構成された古いウェブサイトのレイアウトを、`div`とFlexboxで動的に再構成して表示します。

## 使い方

1. `index.html`をブラウザで開きます。
2. 入力欄に閲覧したいサイトのURLを貼り付けます。
3. 「ページを処理」ボタンを押します。
4. 下のフレームに、安全に処理されたページが表示されます。
